
```jsx
import { CPaginationItem } from '@coreui/react'
// or
import CPaginationItem from '@coreui/react/src/components/pagination/CPaginationItem'
```

| Property | Description | Type | Default |
| --- | --- | --- | --- |
| **active** | Toggle the active state for the component. | `boolean` | - |
| **as** | Component used for the root node. Either a string to use a HTML element or a component. | `(ElementType & string)` \| `(ElementType & ComponentClass<any, any>)` \| `(ElementType & FunctionComponent<any>)` | - |
| **disabled** | Toggle the disabled state for the component. | `boolean` | - |
